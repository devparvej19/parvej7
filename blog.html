1. Difference between map, foreach, filter, find?

Ans. In the forEach method, the result is undefined. In the map method, the transformed elements are returned as a new array. In the filter method, the result is an array that contains the elements that satisfy the condition. The element itself that fulfills the condition is returned by find.

2. Difference between local storage, session storage, cookies?

Ans. Local storage is useful for storing data that users will need to access later. The session storage can improve the performance of web applications. The cookie is useful for storing data that does not need to be persistent.

3. What is event loop? and what is the purpose of event loop?

Ans. A callback queue and a call stack are constantly monitored by the event loop. When the call stack is not empty, the event loop waits until it is empty before placing the next function from the callback queue on the stack. Using this event loop, code is executed, events are collected, and events are processed.

4. what is the purpose of "quizTimer" function?

5. What is the purpose of "chooseAnswer" function?

Ans. "chooseAnswer" function is designed to select and display the user's selected answer among a set of options in multiple choice questions.
